# Uinventory

Prereqs: Node.js 20+, npm, and a Supabase project + Resend account for the env vars.

## Install

* Clone repository and cd into with-supabase-app.
* npm Install
* Create a .env file with the Supabase URL and key, and then resend the API key.


## Run

After installation:
* npm run build
* npm run dev

## Test

* npm test - Jest unit tests (tests-unit/)
* npx playwright test - Playwright end-to-end tests (tests-e2e/)
* npm run lint
